what is margin?

Assistant: In this context, **margin** is the markup we add on top of our actual provider cost before charging the user.

**Example:**
- We pay fal.ai **$0.05** for a 4MP upscale (that's the `base_cost`)
- We set a **30% margin** (`margin_type: percentage`, `margin_value: 0.30`)
- User pays: `$0.05 × 1.30 = $0.065`
- We pocket the $0.015 difference

**Why it exists:** For cheap standard models ($0.02–$0.04), we agreed to do **pass-through** (margin = none, user pays exactly what we pay). But for expensive operations like high-res upscaling, we apply a margin so we're not eating costs if provider prices fluctuate.

Three margin types in the design:
- **none** — pass-through, user pays `base_cost` exactly
- **percentage** — user pays `base_cost × (1 + margin_value)`
- **fixed** — user pays `base_cost + margin_value` (e.g., add flat $0.01)
